# Salário Minimo
Esta biblioteca retorna Lista de salarios minimos desde 1994.

## Instalando

```bash
composer require felipefm32/get-salario-minimo
```

## Usando

```php
use EuFelipeMateus\SalarioMinimo;

print_r(SalarioMinimo::getJson());
```
